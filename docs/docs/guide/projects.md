# Saving & Loading Projects

A project is a snapshot of your entire PhantomCast setup. When you save a project, everything is preserved: surfaces, their positions, video assignments, effects, adjustments, masks, motion trails, and draw order. Load a project later to pick up exactly where you left off.

![Settings page](/img/screenshots/ui-settings.png)

## What Gets Saved

When you save a project, PhantomCast stores:

- All surfaces and their type, shape, and control point positions
- Video and image source assignments for each surface
- Effect settings and blend modes
- Brightness, contrast, saturation, and opacity values
- All exclusion masks and their enabled/disabled state
- All motion trails with their paths and appearance settings
- Surface draw order (layering)

## Saving a Project

1. Go to the **Settings** page using the sidebar
2. In the project section, click **Save Project**
3. Enter a name for your project (e.g., "Halloween Front Wall" or "Christmas Display")
4. Click **Save**

Your configuration is stored and can be loaded at any time.

:::tip
Save your project after calibration and before making experimental changes. That way you always have a known-good state to return to.
:::

## Loading a Project

1. Go to the **Settings** page
2. Find your saved project in the project list
3. Click **Load**
4. PhantomCast restores the full configuration

Loading replaces your current setup entirely, so make sure to save any unsaved changes first.

## Exporting a Configuration Bundle

You can download your project as a file to keep as a backup or transfer to another PhantomCast setup:

1. On the **Settings** page, click **Export**
2. A configuration bundle downloads to your computer
3. This file contains your complete project configuration

Use this to:

- **Back up** your work before making big changes
- **Transfer** a configuration to a different PhantomCast installation
- **Share** setups with other users

## Practical Workflow

A good project management habit looks like this:

1. Set up and calibrate your projection
2. Save the project with a descriptive name
3. Experiment with effects, masks, and motions
4. Save again (with a new name if you want to keep both versions)
5. Before a show, load the project you want to use and verify everything looks correct
